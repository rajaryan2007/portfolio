# Minimalist Developer Portfolio

A blazing-fast, highly responsive personal portfolio website built from the ground up using pure HTML, CSS, and Vanilla JavaScript. 

**Live Demo:** [View Portfolio Here](https://portfolio-tau-virid-23.vercel.app/index.html)

---

## Performance First
This project was engineered with a strict focus on speed and minimal payload. 
* **Zero Image Bloat:** The entire visual experience is crafted using code (CSS/JS) to completely eliminate image-loading bottlenecks.
* **Pure Client-Side:** No heavy frameworks or server-side rendering delays.
* **Optimized Bundle:** Runs exclusively on a streamlined JavaScript bundle, ensuring instantaneous load times and a perfect performance score.

## Design & UI
* **Fully Responsive:** Fluid layouts that adapt seamlessly to any device—from large desktop monitors to small mobile screens.
* **Smooth Animations:** Carefully crafted CSS and JavaScript animations that add life to the interface without dragging down the frame rate.
* **Minimalist Aesthetic:** A clean, distraction-free design that puts the focus entirely on the work and content.

## Tech Stack
* **HTML5:** Semantic and accessible markup.
* **CSS3:** Custom styling, flexbox/grid layouts, and native animations.
* **Vanilla JavaScript:** DOM manipulation and interactive elements without the overhead of external libraries.
